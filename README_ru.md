## 🪐 О Проекте
**EDM Data Base** — это легковесный сервер и защищенное хранилище секретных файлов. Позволяет хранить файлы локально (например, на Android через Termux, Raspberry Pi или любом ПК) и получать к ним безопасный доступ через стильный веб-интерфейс.
Все настройки, аккаунты и управление сервером происходят через встроенную **Консоль Администратора** прямо в терминале.
### ✨ Особенности
 * **Полностью офлайн Фронтенд:** Никаких внешних CDN, Google Fonts или сторонних ресурсов. Всё внутри.
 * **Встроенная CLI-консоль:** Запускайте сервер, управляйте доступом и следите за статистикой через красивый терминальный UI (TUI).
 * **Адаптивный Web UI:** Брутализм / Хакерский дизайн, темы (Светлая, Тёмная, Матрица), настройка форм, цветов и тактильной отдачи.
 * **Поддержка Медиа:** Встроенный просмотрщик изображений, видеоплеер и редактор кода/текста прямо в браузере.
 * **Скрытие Хранилища:** Мгновенно добавляет точку (.) к папке, чтобы скрыть её от файловых менеджеров (управляется через UI или консоль).
 * **Mobile First:** Идеально подходит для запуска на Termux (Android) с доступом через браузер телефона.

## 🚀 Установка и Запуск
### Требования
 * Установленный **Python 3.8+**.
 * (Опционально) **Termux**, если вы запускаете сервер на Android.
### 1. Клонирование репозитория
Откройте терминал и выполните команды:
```bash
git clone [https://github.com/ВАШ_НИК/edm-database.git](https://github.com/ВАШ_НИК/edm-database.git)
cd edm-database

```
### 2. Установка зависимостей
Установите необходимые пакеты (требуется только Flask):
```bash
pip install -r requirements.txt

```
*(Если файла requirements.txt нет, просто выполните: pip install flask)*
### 3. Запуск Сервера / Консоли
Запустите интегрированную панель управления:
```bash
python server.py

```
В меню терминала вы можете:
 1. Задать безопасный путь к хранилищу.
 2. Настроить логин (ID) и пароль (KEY).
 3. Нажать [ 1 ] для запуска сервера Flask.
 4. Открыть предоставленный локальный IP в браузере (например, http://192.168.1.X:5000).
## 🛠️ Использование
 * **Загрузка:** Drag & drop или кнопки + FILE / UPLOAD.
 * **Контекстное меню:** Нажмите ⋮ или используйте долгое нажатие (long-press) на папке для переименования, тегирования или удаления файлов.
 * **Настройки:** В панели настроек можно сменить тему, включить Автосохранение или изменить скругления углов.
